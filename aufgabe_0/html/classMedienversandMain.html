Detailed Description

Simulation eines Medienversands der Buecher, CDs und DVDs vertreibt. Es werden fuer 365 Tage eine vom Benutzer gewuenschte Anzahl Medien zufaellig generiert und in die Dateien Medien.dat und Verkauf.dat geschrieben. Es kann danach fuer einen beliebigen Tag die Verkaufsdaten ausgegeben werden. Es handelt sich um ein Terminal Programm.
