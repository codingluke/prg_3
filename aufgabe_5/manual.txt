NAME

       bruch - Berechnet, vergleicht und generiert Brueche.

SYNOPSIS

    Rechnen mit Bruechen
       ./bruch a b op c d
       ./bruch a op c d
       ./bruch a b op c

    Zwei Brueche Vergleichen
       ./bruch a b -v c d

    Zufaellige Brueche generieren und sortieren
       ./bruch n [ a b c d ] +
       ./bruch n [ a b c d ] -


OPTIONS

    a, c
        Integer, Zaehler

    b, d
        Integer, Nenner, duerfen nicht 0 sein.

    op
        +,-,*,/
        WARNUNG: * sollte auf der console mit \* escaped werden.

    n
        Anzahl als Integer zwischen 1 und 10000

    +
        aufsteigend sortiert

    -
        absteigend sortiert

    []
        keine wirkliche bedeutung.
