Entrypoint to the program "Lottery". Lottery is a Program to simulate and analyze lotto winnings. There are following possibilities to choose from.

frequency scale, arrangement of the minima, arrangement of the maxima, arrangement of the lengths, number of winning which has their lengths also as number, number of odd winnings, number of even winnings, number of prim winnings, number of non prim winnings, arrangement of k-street winnings, tip a LotteryWin, automated tipping and generate a neu number of winnings.

The program has to be started with one extra parameter including the number of winnings to generate.

./Lottery 5000000
